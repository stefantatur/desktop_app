# Desktop Task Manager

Небольшое десктопное приложение на Python с использованием `flet`.

Проект сделан для предмета "Объектно ориентированное программирование".
В приложении можно добавлять задачи, задавать категорию и приоритет,
искать задачи, фильтровать их по статусу, отмечать выполненными и удалять.
Данные сохраняются в локальный файл `tasks.json`.

## Использованные классы

- `Task` - модель одной задачи.
- `Priority` - перечисление уровней приоритета.
- `TaskStatistics` - подсчет статистики по задачам.
- `TaskCreateSchema` и `TaskFileSchema` - Pydantic-классы для проверки данных.
- `TaskRepository` - абстрактный класс хранилища.
- `JsonTaskRepository` - конкретная реализация хранилища через JSON.
- `TaskService` - бизнес-логика приложения.
- `TaskApp` - графический интерфейс приложения.
- `ApplicationFactory` - сборка приложения и зависимостей.

## Особенности ООП

- проект разделен на `logic` и `infra` слои;
- в `logic` находятся сущности, сервис и абстрактный репозиторий;
- в `infra` находятся конкретные реализации: JSON-хранилище, GUI и factory;
- бизнес-логика не зависит от интерфейса и формата хранения;
- для приоритетов используется `Enum`;
- для статистики используется отдельный `dataclass`;
- Pydantic используется для валидации ввода и данных из JSON;
- для создания приложения используется factory.

## Интерфейс

Интерфейс сделан через библиотеку `flet`.
Она используется для создания desktop-приложений с графическим интерфейсом:
кнопками, полями ввода, выпадающими списками и списком задач.

Это именно desktop-приложение: Flet открывает отдельное окно,
имеет графический интерфейс и может быть собрано в macOS-приложение `.app`.

## Как запустить

### Запуск из исходников

Нужен Python 3.10 или новее.

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить приложение:

```bash
python3 main.py
```

### Сборка .app

```bash
pip install -r requirements.txt
flet pack main.py --name TaskManager
```

После сборки приложение появится в папке `dist/`.

## Что можно делать

- добавить новую задачу;
- указать категорию;
- выбрать приоритет;
- найти задачу по названию или категории;
- отфильтровать задачи по статусу;
- отметить задачу выполненной или активной;
- удалить задачу;
- очистить все выполненные задачи;
- закрыть приложение и открыть снова без потери списка задач.
